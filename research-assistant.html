Linguistics Lab Research Assistant

This lab was focused on studying the differences in linguistics comprehension between native Spanish speakers and learned Spanish speakers.

Project Goals

The main goal of this project was to gather data from Spanish bilingual speakers in order to determine whether there were differences in linguistic processing concerning gendered nouns. This project was interested in data from child speakers, as their learning capabilities may make the data more insightful.
